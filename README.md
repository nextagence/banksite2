# NovaPay Demo (GitHub Pages ready)

Это статический демо-сайт: лендинг + инвест-питч + кабинет (читает мок-данные из /api).

## Быстрый старт локально
В папке проекта:
```bash
python3 -m http.server 8000
```
Открыть:
- http://localhost:8000/landing.html
- http://localhost:8000/deck.html
- http://localhost:8000/app.html

## Деплой на GitHub Pages
1) Залей репозиторий
2) Settings → Pages → Deploy from branch (main / root)
3) Открывай:
- /landing.html
- /deck.html
- /app.html

## Лиды
В этом архиве отправка лидов отключена (без Telegram / backend).
